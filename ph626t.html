Overview

H626T is the new generation High Definition and High Performance IPTV/OTT DVB-T Hybrid Set-Top Box as the second STB for family. This platform were designed into new elegant compact appearance. Equipped with new generation processor, this platform support HEVC, video decoding maximum support to 1080p, as well as support advanced audio format decoding, including Dolby and Dolby plus. These attractive features could fully satisfies FetchTV demands for the second Set-Top Box.

The platform offers customization services, including remote control, box packing, logo and user manual. Besides, this platform has been integrated Hybroad Archimage middleware and FetchTV middleware. FetchTV could use its own servers to launch IPTV/OTT service.

Features:

Benefits for Operators/ISPs:

-High Performance : New generation B15 Apps CPU.

-Attractive features : Video decoding maximum support to 1080p. Support HEVC. Integrated a DVB-T tuner and 802.11ac Wi-Fi module.

-High Security : Integrated Verimatrix Advance Security CAS.

-Seamlessly fits to existing end-to-end system : Hybroad H626T provides an easy-to-setup and flexible means of IPTV/OTT deployment.
