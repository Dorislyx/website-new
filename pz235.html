Overview

High-definition television is now a general feature of in-room services. The Hybroad Set-Top Box Z235 is a high-definition STB providing a comprehensive range of features for the hospitality market. Fully functionalities and advanced Hardware, Z235 support decoding all range of video format, as well as advanced audio format.

The platform offers attractive features for operators/ISPs. One of the attractive features is its IR_OUT interface, which support external infrared emitter. The next attractive feature is IR_IN interface, which support external infrared receiver. Equipped with a RJ12 serial interface that can be used as a connection interface to hospitality televisions for use with a single remote control. Two RJ45 interfaces, which can be used as a switchboard, one for STB, the other for a peripheral computer device.

Features:

Benefits for Operators/ISPs:

-High Performance : Leading dual-core processor solution.

-Attractive features : Video decoding maximum support to 1080p. Support IR_OUT and IR_IN interface, a RJ12 interface and two RJ45 interface.

-High secure : Integrated Verimatrix High Security CAS.

-Perfect Hotel IPTV/OTT Set-Top Box solution : Hybroad Z235 provides an easy-to-setup and flexible means of Hotel IPTV/OTT deployment.
