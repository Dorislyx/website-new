Overview

TV de alta definição é agora uma característica geral dos serviços de quarto. O Hybroad Set-Top Box Z235 é um STB de alta definição fornecendo uma gama abrangente de recursos para o mercado de hospitalidade. Com funcionalidades completas e hardware avançado, Z235 suporta decodificar todos formatos de vídeo, bem como o formato de áudio avançado.

A plataforma oferece detalhes atrativos para os operadores/ISPs. Uma das características atraentes é a sua interface IR_OUT, que suporta externo emissor infravermelho, ligar e desligar da TV e outro STB. Equipado com uma RJ12 interface serial que pode ser usado como uma conexão entre interface e a televisões ,controlaso por um único controlo remoto. Duas RJ45 interfaces,que podem ser utilizadas como um painel de comando, um para STB,e o outro para dispositivos periféricos de computador .

Features:

Benefícios para os Operadores / ISPs:

-Alto Desempenho: solução avançada dual-core processador.

-Características atraentes: decodificação de vídeo suporta 1080p ao máximo. Suporta interface de IR_OUT e IR_IN, uma interface RJ12 e duas interface RJ45.

-Alta Segurança: Integrado Verimatrix Alta Segurança CAS.

-Solução de Hotel perfeito IPTV / OTT Set-Top-Box: Hybroad Z235 oferece um easy-to-setup e meios flexíveis de implantação de Hotel IPTV / OTT.
